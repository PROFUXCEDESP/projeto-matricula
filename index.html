3. Tecnologias & Design

Fonte Oficial: Sofia Pro

Esta é a fonte da marca. Se não tivermos a licença web, usaremos a Inter como fallback.

font-family: 'Sofia Pro', 'Inter', sans-serif;

Ícones: Lucide React

Não usaremos Google Icons. Usaremos Lucide (mais leve e moderno).

Instalação: npm install lucide-react

Exemplo de uso:

import { Camera, User } from 'lucide-react';
<Camera color="red" size={48} />

4. Setup e Git Seguro

⚠️ REGRA DE OURO DO GIT:
Equipe Mobile mexe SÓ na pasta /mobile.
Equipe Desktop mexe SÓ na pasta /desktop.
Isso evita conflitos!

Passo a Passo (Instalação)

  1. Baixar e Instalar Node.js (LTS).
  2. Baixar e Instalar VS Code.
  3. Baixar e Instalar Git.
  4. Criar pasta do projeto e abrir no VS Code.
  5. Rodar no terminal: npm install
  6. Rodar o projeto: npm run dev
